General configuration options

These are the general configuration options.

Check for updates

Whether the plugin will check for a newer version. Version checking just reads a text file from here.

check_for_updates: [true|false]

The default is true.

TARDIS key

Set the material used as the TARDIS key. Use any valid Bukkit Material ENUM.

key: [Material]

The default is STICK.

Give key

This works in conjunction with a multi-world inventory plugin, and sets whether the TARDIS key item is given to players when they time travel to another world (in which they don’t have the key in their inventory).

give_key: [true|false]

The default is false.

Bonus chest

Toggle whether the TARDIS chest is filled with items replaced during the TARDIS construction.

bonus_chest: [true|false]

The default is true.

Max teleport radius

You can set the maximum distance (in blocks) that players can teleport to in their TARDIS

tp_radius: [x]

Change the [x] to the desired number of blocks. The default is 256.

TARDIS platform

Whether the TARDIS platform should be enabled for the plugin, this overrides player platform settings

platform: [true|false]

The default is false.

Sound Effects

Whether sounds effects should be enabled for the plugin, this overrides player SFX settings

sfx: [true|false]

The default is true.

Landing on water

Whether the TARDIS will land on water. If set to true, it is recommended that platform is set to true.

land_on_water: [true|false]

The default is true.

Name the TARDIS

Whether the sign on the TARDIS Police Box has the player’s name on the first line.

name_tardis: [true|false]

The default is false.

TARDIS Police Box materialisation

Whether the TARDIS Police Box materialises when it lands. It takes longer than normal but looks cooler :)

materialise: [true|false]

The default is true.

The Chameleon Circuit

Whether the TARDIS Police Box can try to match its surroundings by changing the wall block.

chameleon: [true|false]

The default is true.

You can also restrict the use of TNT and precious blocks (such as IRON, GOLD, EMERALD and DIAMOND) being used for Police Box walls (and room walls and floors).

all_blocks: [true|false]

The default is false.

Mob farming

Whether you can bring farm animals into the TARDIS if they are captured in the Police Box.

allow_mob_farming: [true|false]

The default is true.

Autonomous homing function

Whether players can use the autonomous homing function preference.

allow_autonomous: [true|false]

The default is true.

TARDIS deletion confirmation time

Set the time in seconds, that the delete TARDIS confimation command (/tardis exterminate) must be typed within.

confirm_timeout: [seconds]

The default is 15.

WorldGuard

Whether WorldGuard should be used to protect the inner TARDIS, and whether travelling respects WorldGuard regions (if players do not have rights to build in an area, they will be denied access)

use_worldguard: [true|false]
respect_worldguard: [true|false]

The default is true.

Towny

Whether travelling respects Towny regions (the TARDIS will only land in Wilderness)

respect_towny: [true|false]

The default is true.

WorldBorder

Whether travelling respects WorldBorder limits (the TARDIS will not travel beyond the border)

respect_worldborder: [true|false]

The default is true.

Random location finding timeout

Set the maximum number of seconds the random location finder is allowed to take, and the height the random destination defaults to if the random location finder does time out

timeout: [seconds]
timeout_height: [y-height]

The defaults are 5 and 135

The configuration file (config.yml) is generated automatically, you can find it at plugins/TARDIS/config.yml

All configuration options can be set via in game commands or in the console, see the Admin Commands page.

TARDIS key

Set the material used as the TARDIS key. Use any valid Bukkit Material ENUM.

key: [Material]

The default is STICK.

Bonus chest

Toggle whether the TARDIS chest is filled with items replaced during the TARDIS construction.

bonus_chest: [true|false]

The default is true.

Max teleport radius

You can set the maximum distance (in blocks) that players can teleport to in their TARDIS

tp_radius: [x]

Change the [x] to the desired number of blocks. The default is 256.

Protect the TARDIS

You can set whether the TARDIS is fire and lava proof. With the current release this affects GREY, LIGHT GREY, BLUE and ORANGE wool server wide, unless overridden by another plugin (Listener Priority = LOW).

protect_blocks: [true|false]

The default is true.

Default world

Set whether the (inner) TARDIS is constructed in a specific world (for instance a FLAT world).

default_world: [true|false]

The default is false.

Default world name

Set the default world that the TARDIS is constructed in when default_world is true.

default_world_name: [World]

The default is myridiculouslylongworldnameiscalledcuthbert

Include default world

Toggle whether the default world is included in the possible destinations when time traveling in the TARDIS.

include_default_world: [true|false]

The default is false.

TARDIS platform

Whether the TARDIS platform should be enabled for the plugin, this overrides player platform settings

platform: [true|false]

The default is false.

Sound Effects

Whether sounds effects should be enabled for the plugin, this overrides player SFX settings

sfx: [true|false]

The default is true.

The End

Whether travel to End worlds should be enabled for the plugin

the_end: [true|false]

The default is false.

The Nether

Set whether NETHER worlds are included in random Time Travel destinations

nether: [true|false]

The default is false.

WorldGuard

Whether WorldGuard should be used to protect the inner TARDIS, and whether travelling respects WorldGuard regions created by the WorldGuard other plugins (if players do not have rights to build in an area, they will be denied access)

use_worldguard: [true|false]
respect_worldguard: [true|false]

The defaults are true.

Random location finding timeout

Set the maximum number of seconds the random location finder is allowed to take, and the height the random destination defaults to if the random location finder does time out

timeout: [seconds]
timeout_height: [y-height]

The defaults are 5 and 135

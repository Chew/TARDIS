Installation

Get the TARDIS plugin files from: http://dev.bukkit.org/server-mods/tardis

You install TARDIS just like you would any other Bukkit plugin, by dropping it into the Bukkit plugins folder and starting the server. The TARDIS database and configuration files will be created automatically.

While the TARDIS plugin is backwards compatible with previous versions of CraftBukkit (and is compatible with Tekkit Classic Server), for the plugin to have full functionality and work correctly you should have CraftBukkit version 1.4.7-R1.0 or higher.

Note: There is a bug with Spigot, FTB Mindcrack and MCPC+ server variants that causes the TARDIS to be built with blocks missing - therefore these servers are NOT supported (pull requests for fixes gratefully accepted).

Clean Installation:

Place the following jar files in your plugins folder:

TARDIS.jar
TARDISChunkGenerator.jar

Upgrade Installation:

Backup your TARDIS Folder!

Place the upgraded jar files in the plugin folder.

Upgrade notes:

It is recommended to delete the TARDIS schematics folder before an upgrade!

If you have trouble after an upgrade, please try a fresh install of the plugin by deleting your TARDIS Folder.

Enhancing the TARDIS plugin

While not essential to make the TARDIS plugin work, the following plugins will make your Time Travelleing experience even better.

  1. Install Multiverse so you have multiple worlds to travel to. This is also required if you want to generate TARDISes in their own self-conatined world.
  2. Install the TARDISChunkGenerator plugin so that you can generate TARDISes in their own self-conatined world.
  3. Install the SpoutPlugin, so that you can hear TARDIS and Doctor Who sound effects. Note: You will also need to connect to the server with the SpoutCraft client.
  4. Install WorldGuard to automatically protect your TARDIS from griefing, and stop mobs spawning in TARDIS worlds.
  5. Install WorldBorder to restrict the size of the worlds generated for TARDIS occupation.
  6. Install a Permissions plugin, so that you can restrict or grant players the rights to use TARDIS’ different features. We like PermissionsBukkit.

Install Notes:

If you desire your TARDIS to have its own world, please have Multiverse or another multi-world plugin installed and set create_worlds: true in your TARDIS configuration file.

If upgrading from a previous version of TARDIS that did not have generated worlds, or enabling them for the first time, please delete your previously created TARDIS and recreate them.

If upgrading from a version without an Artron Energy button or handbrake, please place these items, and use the /tardis update command to add them to your TARDIS. See Commands for more information.

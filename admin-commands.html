Admin commands

/tardisadmin

To view the current TARDIS plugin configuration type:

/tardisadmin config

To view specific TARDIS plugin configuration otions:

/tardisadmin config [worlds|rechargers|rooms]

Most TARDIS plugin config options can be set ingame or via the console, and take effect immediately.

There are three types of config options:

  1. Boolean values e.g. true or false
  2. String values e.g. the names of things, like the TARDIS key
  3. Integer values e.g. numbers, like the cost of a room

To use the /tardisadmin command to set config options, you use the syntax:

/tardisadmin [config option] [value]

For example, to set the make the TARDIS never land on water you would type:

/tardisadmin land_on_water false

Boolean config options:

add_perms — set whether a TARDIS world is assigned permissions when the server uses a permissions plugin that has per-world configuration — see Add permissions.

all_blocks — set whether the chameleon circuit is allowed to use precious blocks such as GOLD and DIAMOND.

allow_autonomous — set whether players can use the autonomous function.

allow_hads — set whether the TARDIS can escape hostile actions.

allow_mob_farming — set whether players can farm mobs when entering the TARDIS.

allow_tp_switch — set whether players can set and switch to texture packs when entering and exiting the TARDIS.

bonus_chest — set whether the TARDIS chest is filled with items replaced during the TARDIS construction.

chameleon — set whether the TARDIS police box can change its appearance to match its surroundings.

check_for_updates — whether to have the plugin check for an updated version. Version checking just reads a text file from here

create_worlds — set whether TARDIS are created in their own separate worlds. Requires the TARDISChunkGenerator plugin and a multi-world plugin.

default_world — set whether to use a default world for (inner) TARDIS construction.

emergency_npc — whether the Emergency Program One feature is enabled.

exile — whether the TARDIS exile feature is enabled.

give_key — sets whether the TARDIS key is given when changing worlds (and using a multi-world inventory plugin).

include_default_world — set whether the default world is included in time travel destinations.

land_on_water — set whether the TARDIS will land on water in the Overworld.

materialise — set whether the TARDIS police box simulates ’materialising’ upon entering a new time travel destination.

name_tardis — set whether to put the player’s name on the Police Box sign.

nether — set whether the TARDIS is allowed to travel to Nether worlds.

per_world_perms — whether players require the tardis.travel.[world] permission to travel to a world.

platform — set whether a platform is created outside the Police Box door if the exit is located over AIR or WATER — this overrides user preferences if turned off.

respect_factions — same as respect_worldguard, but for the Factions plugin.

respect_towny — same as respect_worldguard, but for the Towny plugin.

respect_worldborder — same as respect_worldguard, but for the WorldBorder plugin.

respect_worldguard — set whether time travelling respects protected WorldGuard regions (if players do not have rights to build in an area, they will be denied access).

return_room_seed — set whether the room seed block is returned to the player when they jettison the room.

rooms_require_blocks — set whether growing a room requires the player to condense the materials required to grow the room in the TARDIS condenser.

sfx — set whether sound effects are played in the TARDIS interior — this overrides user preferences if turned off.

spawn_eggs — set whether the player recieves mob spawn eggs when TARDIS mob farming (and there is no farm room grown).

the_end — set whether the TARDIS is allowed to travel to The End worlds.

use_worldguard — set whether the TARDIS interior becomes a protected WorldGuard region belonging to the player who created the TARDIS.

String values

key — set the TARDIS key item. You can use any valid Bukkit Material ENUM. A list can be found here: http://jd.bukkit.org/apidocs/org/bukkit/Material.html

full_charge_item — set the item that when clicked on the TARDIS Artron Energy button, boosts the Capacitor to 100%.

default_world_name — set the name of the default world. This is only used if default_world is true.

gamemode — set the gamemode of TARDIS worlds when they are created.

inventory_group — If the Multiverse-Inventories plugin is enabled on the server, you can add the group that TARDIS worlds are added to when they are created.

Integer values

border_radius — sets the distance of the WorldBorder barrier (only used if the plugin is installed on the server).

confirm_timeout — set the number of seconds the player has to type the /tardis exterminate command when destroying a TARDIS.

count — set the maximum number of times a player can build and destroy a TARDIS.

gravity_max_distance — sets the maximum distance a gravity well can be set to.

gravity_max_velocity — sets the maximum velocity a gravity well can be set to.

hads_damage — sets the number of hits the Police Box will take before engaging HADS.

hads_distance — sets the distnace in blocks the Police Box will travel when HADS is engaged.

full_charge — sets the Artron Energy Capacitor’s full amount.

jettison — set the percentage of the original cost of a room you get back when jettisoning it.

lightning_recharge & creeper_recharge — set the amount of energy gained form natural lightning strikes and given to players for killing a charged creeper.

malfunction_end — set the percentage chance a malfunction will deposit the TARDIS in the The End.

malfunction_nether — set the percentage chance a malfunction will deposit the TARDIS in the Nether.

malfunction — set the percentage chance the TARDIS will malfunction when travelling.

nether_min & the_end_min — set the minmum Artron Energy level required to travel to the Nether and The End.

player — sets the amount of Artron Energy a player receives each time they time travel. If the Minecraft/CraftBukkit version does not include beacon blocks, then this amount is multiplied by 10)

random & travel & comehere & hide — sets the Artron energy costs for each of these forms of time travel.

recharge_distance — the maximum distance a TARDIS can be from a beacon in order to recharge. This is also the distance from a natural lightning strike, for the TARDIS to gain bonus energy.

timeout & timeout_height — sets the maximum time a random location task can take. If the task timesout then the Police Box location is set at the timeout_height value. This prevents the plugin crashing when using skyblock type worlds.

tp_radius — set the maximum distance (in blocks) you can time travel in the TARDIS. Note The actual values is about 4 times this (due to the multiplier repeater).

Reloading the config

If you have made changes to the config file by manually editing it, you should use the reload command to reload the changes. Type:

/tardisadmin reload

Note: Do not use the server /reload command or the manual changes you have made will be overwritten when the TARDIS config is saved when it is disabled.

Other admin commands

Enter a player’s TARDIS (by direct teleport instead of the door).

/tardisadmin enter [player]

Set the beacon block you are targeting in game, as a TARDIS recharge station.

/tardisadmin recharger [name]

Remove the specified beacon recharger from the config.

/tardisadmin decharge [name]

Set whether a specified world is included or excluded from random time travel destinations.

/tardisadmin include [world]
/tardisadmin exclude [world]

List all TARDISs and their locations.

/tardisadmin list

Remove a specified player’s TARDIS.

/tardisadmin delete [player]

List TARDISes that haven’t been used for an extended period.

/tardisadmin prune list [number of days]

Remove TARDISes that haven’t been used for an extended period. See the Prune page for more details.

/tardisadmin prune [number of days]

Check or set a player’s TARDIS count.

/tardisadmin playercount [player]
/tardisadmin playercount [player] [count]

List chunks that are being kept loaded by TARDIS — either for room growing or Police Box locations.

/tardisadmin chunks

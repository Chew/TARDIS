Planet configuration options

These are the plugin planet configuration options (found in planets.yml).

Option Type Default Value
switch_resource_packs boolean false
  To make the Skaro world look a little different than a regular world, TARDIS can set a small (79KB) resource pack when you switch worlds (you must enable server resource packs in the Minecraft client). The resource pack changes the colour of the sky, clouds, water, lava and fog — you will need to install Optifine
set_pack_on_join boolean true
  If players join the server in the Skaro world, setting this to true will switch the resource pack for them
default_resource_pack string https://dl.dropboxusercontent.com/u/53758864/rp/ Default.zip
  This is just an empty resource pack (954 bytes). It is needed to revert back to the default (or other installed) resource pack colours. You can either host your own, or use the one provided. Worlds not listed in the planets section will use this
planets:
  This is a list of worlds on your server. If you want to use the world resource pack switching feature for other server worlds, you can add them here. Use the TARDIS_TimeVortex entry as an example.
    Skaro:
        enabled boolean false
  Whether this world should be created and managed by TARDIS
        resource_pack string https://dl.dropboxusercontent.com/u/53758864/rp/ Skaro.zip
  This is where you specify the URL to the resource pack you want to switch to when entering the world. Host your own, or use the one provided
        acid boolean true
  Whether all the water in the world is acid (and harms the player)
        acid_damage integer 5
  The amount of damage a player takes when in acid water
        acid_potions string list - WEAKNESS
- POISON
  Optional list of potion effects to give the player when they are in acid water. You can add any potion types from the list here: SPIGOT PotionType.java. Set to [] to remove potion effects
        rust boolean true
  Whether rust is enabled (not actually used yet)
    TARDIS_TimeVortex:
        resource_pack string default
  Sets the resource pack for this world

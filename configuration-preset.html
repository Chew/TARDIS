Police Box & preset configuration options

These are the Police Box & preset configuration options.

Option Type Default Value
police_box:
    name_tardis boolean false
  Sets whether to put the player’s name on the Police Box sign.
    wall_id integer 35
  Sets the default material of the Police Box walls. Only applicable if use_block_stack: true
    wall_data integer 11
  Sets the default damage value of the Police Box walls. Only applicable if use_block_stack: true
    tardis_lamp string REDSTONE_LAMP_OFF
  Sets the default material of the Police Box lamp. Only applicable if use_block_stack: true
    materialise boolean true
  Sets whether the TARDIS runs the materialising animation when it takes off and lands.
    confirm_timeout integer 15
  Sets the number of seconds the player has to type the /tardis exterminate command when destroying a TARDIS.
    sign_colour string WHITE
  Sets the colour of the text on a TARDIS sign. Must be a valid ChatColor value, see https://hub.spigotmc.org/javadocs/spigot/org/bukkit/ChatColor.html
    set_biome boolean true
  Whether to set the blocks the TARDIS lands on to DEEP_OCEAN biome. This setting allows the custom TARDIS block textures to work.
    default_preset string FACTORY
  Sets the default preset that appears when the TARDIS is first created. See the Chameleon Presets page for values that can be set.

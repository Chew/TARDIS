Player Preferences

There are four specific player preferences that you can change:

  • Safety platform
  • Sound effects
  • Who quotes
  • Wall material

Each preference can be set with a command.

Player preference commands

/tardisprefs

To toggle the TARDIS safety platform on or off, type:

/tardisprefs platform [on|off]

If the server admin has disabled the safety platform in the plugin config, this command will have no effect.

To toggle the TARDIS sound effects on or off, type:

/tardisprefs sfx [on|off]

If the server admin has disabled sound effects in the plugin config, this command will have no effect.

To toggle the Who quotes on or off, type:

/tardisprefs quotes [on|off]

To change the block room walls are grown with, type:

/tardisprefs wall [material]

Where [material] is a block from the list on the Wall materials page. Type /tardisprefs wall help to see the list in-game.

To change whether the TARDIS Autonomous homing function takes the Police Box to the nearest recharge point or the Tardis ‘home’ (whichever is closest) when the Timelord owner dies, type:

/tardisprefs auto [on|off]

If the server admin has disabled the autonomous homing function in the plugin config, this command will have no effect.

Other preferences

/tardis chameleon

You can toggle the Chameleon Circuit on or off with this command, type:

/tardis chameleon on
/tardis chameleon off

The Bind command

To make travelling to saved destinations easier you can ‘bind’ them to buttons, levers and signs.

You can also bind the following commands:

/tardis hide
/tardis rebuild
/tardistravel home

Binding

To bind the destination/command to a block run the command:

/tardis bind [save name]
    or
/tardis bind [hide|rebuild|home]

Follow the onscreen instructions, and click the block you want to bind.

Unbinding

To unbind a destination/command fromo a block run the command:

/tardis bind [save name or command]

You will receive a confirmation message that the block is now safe to be removed.

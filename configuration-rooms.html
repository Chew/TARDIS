Room configuration section

The rooms configuration section sets the seed block and Artron Energy costs for growing rooms. Each room type has its own sub-section with two keys: the room cost, and the room seed.

The format is:

rooms:
   [ROOMTYPE]:
      cost: [amount]
      seed: [MATERIAL]

The defaults are:

ROOMTYPE cost seed block
ARBORETUM 325 LEAVES
BAKER 350 ENDER_STONE
BEDROOM 475 GLOWSTONE
EMPTY 250 GLASS
GRAVITY 625 MOSSY_COBBLESTONE
ANTIGRAVITY 625 SANDSTONE
HARMONY 450 SMOOTH_STAIRS
KITCHEN 450 PUMPKIN
LIBRARY 550 BOOKSHELF
PASSAGE 325 LEAVES
POOL 450 SNOW_BLOCK
VAULT 350 DISPENSER
WOOD 350 WOOD
WORKSHOP 400 NETHER_BRICK

Room jettisons

Set the percentage of energy the TARDIS gets back when a room is jettisoned.

jettison: [percentage]

The default is 75.

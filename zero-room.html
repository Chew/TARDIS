TARDIS Zero Room

The Zero Room is one of the most peaceful locations in existance. It is a place completely cut off from the rest of the universe, cut off even from the TARDIS interior, in a entirely separate dimension.

To create a Zero Room, simply run the command /tardis room ZERO. If you have condensed the required material

Note: The Zero room must be grown using the command, as it is not part of the ARS.
